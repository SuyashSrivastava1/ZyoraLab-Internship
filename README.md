## 🏃 How to Run the Launch Page Locally

The Developer Launch Page (`index.html`) is built using **Vanilla HTML5, CSS3, and ES6+ JavaScript** for zero-dependency maximum performance.

### Option 1: Direct File Execution
Simply open `index.html` directly in your web browser (`Double-Click` or drag-and-drop into Chrome / Edge / Firefox).

### Option 2: Live Server (Recommended for Hot Reloading)
If you have Node.js installed, you can spin up a local development server:
```bash
# Run using npx and serve
npx serve .
# Or using live-server
npx live-server .
```
Access the application at `http://localhost:3000` (or the port indicated in terminal).
